# Simple example using GraphQL with Express/React


Start
-----

Fill in the data in .env files

`backend/.env` 
```dotenv
PORT=5000
```

`frontend/.env` 
```dotenv
REACT_APP_BACKEND_URL=http://localhost:5000/graphql
```

Run
---

```
docker-compose up -d
```

### Standard ports

* [Backend](http://localhost:8000/): 8000
* [Frontend](http://localhost:3001/): 3000
